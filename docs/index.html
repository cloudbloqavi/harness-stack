An AI-native engineering harness
for any repo, any coding agent.

harness init reads your project and generates spec-driven sub-agents — wired for Claude Code, Cursor, Codex, Antigravity, or Copilot — as diffable YAML you can audit, edit, and commit.

$ npm install -g git+https://github.com/cloudbloqavi/harness-stack.git
